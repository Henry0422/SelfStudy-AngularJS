Example 18 - JSON Filter

The JSON filter will take a JSON, or JavaScript object, and turn it into a string.

Code:
{{ {'name': 'Ari', 'City': 'San Francisco'} | json }}

Result:
{{ {'name': 'Ari', 'City': 'San Francisco'} | json }}
